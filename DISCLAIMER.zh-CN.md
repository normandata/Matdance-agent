# 声明

Language: [English](DISCLAIMER.md) | 中文

Matdance 是一个本地优先的 agent 运行时，不是云端托管服务、企业权限平台、法律/医疗/金融建议系统，也不是隐私保险柜。

## 模型行为

Matdance 会通过 prompt、工具描述、Settings 权限和 host 侧限制降低误操作风险，但模型输出仍然来自你配置的上游模型。不同模型的指令遵循、安全边界、工具调用稳定性和多模态能力差异很大。项目不能保证模型永远正确、永远遵守约束，或者永远不会受到外部文本诱导。

## 隐私数据

隐私访问开关是系统给 agent 的实时权限信号。开关关闭时，理想情况下 agent 应拒绝访问桌面、照片、私有文档、社交平台、邮箱、私信、论坛账号页等隐私内容。开关开启时，也不代表 agent 可以泄露密码、token、cookie 明值、私密原文或其它高风险内容。

真正敏感的数据应由用户自行访问、筛选、脱敏，再交给 agent。不要把 Matdance 当作替你处理全部隐私边界的代理人。

## 浏览器、cookie 和第三方服务

浏览器自动化基于 Playwright。Matdance 可以保存、列出和应用 cookie，但不会把 cookie 明值返回给用户或模型。cookie 写入浏览器 context 不等于页面一定已经登录，也不等于第三方服务接受该登录态。

连接第三方模型、图像生成、TTS、STT、搜索、网页和站点账号时，你需要自行理解并遵守对应服务的条款、费用、速率限制和数据处理规则。

## 系统稳定性边界

Matdance 源码、插件源码、`.matdance/state`、Web 鉴权状态、supervisor 状态、shadow 运行目录、运行队列、任务运行记录、cookie store、agent 配置、模型凭据、API key、token、密码和授权文件属于系统稳定性边界。不要让 agent 作为媒介修改这些内容。

## 平台支持

当前主要面向 Windows 和 macOS。Linux 做了兼容处理，但不同发行版、桌面环境、shell、权限策略和浏览器依赖可能存在差异。移动设备、边缘设备和异架构设备不属于正式支持范围。
